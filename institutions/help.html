ü§î Frequently Asked Questions

How do I add a new academic program?

To add a new program, navigate to the Programs section and click "Create Program". Fill in the program details, requirements, and deadlines. Make sure to set the capacity and application requirements before publishing.

Can I bulk process applications?

Yes! Select multiple applications using the checkboxes, then use the bulk actions toolbar to accept, reject, or schedule interviews for selected applicants simultaneously.

How do I generate admission reports?

Go to the Reports section and choose from pre-built templates or create custom reports. You can filter by date range, program, status, and export in various formats (PDF, Excel, CSV).

What browsers are supported?

Flow works best with modern browsers: Chrome 90+, Firefox 88+, Safari 14+, and Edge 90+. For optimal performance, keep your browser updated.
